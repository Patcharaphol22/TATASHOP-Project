🛠️ ขั้นตอนการติดตั้งระบบ (Installation Steps)
1. Environment Check: เตรียม Node.js (v16+) และ MongoDB (Local หรือ Atlas) ให้พร้อม

2 Dependencies: รันคำสั่ง npm install และ npm audit fix เพื่อติดตั้ง Library

3 Database Config: ตั้งค่า Connection String ในไฟล์ app.js หรือ .env ให้เชื่อมกับฐานข้อมูลของคุณ

4 System Start: รันระบบด้วยคำสั่ง npm start

หมายเหตุ: หากติด Permission ให้ใช้คำสั่ง Set-ExecutionPolicy ก่อน

5 Web Access: เข้าใช้งานผ่าน Browser ที่ URL: localhost:8080
